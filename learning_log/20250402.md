# 첫 태그 사용하기
## 태그 정의
tag - '이 요소는 어떠한 특징을 지니는 부분이다' 라면서 웹을 구성하는 요소가 각 요소의 역할을 정의하거나 관련 정보를 알려줄 대 사용하는 '꼬리표'

### 여는 태그 vs. 닫는 태그
문법
```html
<여는태그> 내용 </닫는태그>
```

1. 여는 태그와 닫는 태그 사이의 내용은 여러 줄일 수 있는데, 그 사이에는 들여쓰기를 하는 것을 원칙으로 함
2. 닫는 태그가 없어 단독으로 쓰이는 경우도 있음
